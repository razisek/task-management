## Installation
### Clone Project
```sh
git clone https://github.com/razisek/task-management
```
### Change Directory to Project
```sh
cd task-management
```
### Install Dependency
```sh
composer install
```
```sh
npm install
```
### Setting on .env / copy from .env.example\
run migration
```sh
php artisan migrate
```
### Generate a secret key for JWT
```sh
php artisan jwt:secret
```
### Start the server
Laravel Server
```sh
php artisan serve
```
Vite Server
```sh
npm run dev
```
## Build React
run this command to build react

```sh
npm run build
```
